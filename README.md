# Projektarbete Freaky Fashion


Detta är ett skolprojekt där jag har byggt en responsiv webbshop med HTML och CSS.

Projektet innehåller:
- Startsida med produkter
- Produktsida med detaljer
- Kassasida med formulär
- Adminsida för att lägga till produkter

Fokus har legat på:
- Responsiv design (mobil, tablet, desktop)
- CSS Grid och Flexbox
- Struktur och semantisk HTML

Sidan är byggd mobile-first och anpassar sig efter olika skärmstorlekar.
